# FLIPKART-SALES-INSIGHTS
📊 Flipkart Sales Insights Dashboard
📝 Project Overview

This project presents an interactive Sales Insights Dashboard built using Power BI to analyze sales data from Flipkart. The dashboard provides a comprehensive view of product performance, category trends, payment preferences, and time-based sales patterns.

It enables stakeholders to make data-driven decisions by transforming raw sales data into meaningful insights.

🎯 Problem Statement

E-commerce platforms like Flipkart generate massive amounts of sales data daily. However, extracting actionable insights from this data can be challenging without proper visualization tools.

The goal of this project is to:

Analyze sales performance across different categories
Understand customer purchasing behavior
Identify trends based on time and payment methods
Provide an easy-to-use dashboard for quick decision-making
💡 Solution & Features
🔹 Key Dashboard Features
Total Sales by Category
Compare performance across categories like Electronics, Clothing, Beauty, etc.
Sales by Payment Method
Analyze customer preferences (UPI, Credit Card, Debit Card, COD, Wallet, Net Banking)
Time-Based Sales Analysis
Yearly, Quarterly, Monthly, and Daily trends
Product-Level Insights
Track top-selling products and quantities sold
Interactive Filters
Category-wise filtering
Drill-down capabilities for detailed analysis
Dynamic Visualizations
Bar charts, donut charts, and time-series graphs

📂 Dataset Description

The dataset includes the following attributes:

Order ID – Unique identifier for each order
Product Name – Name of the product
Category – Product category
Price (INR) – Price per unit
Quantity Sold – Number of units sold
Total Sales – Total revenue generated
Order Date – Date of purchase
Payment Method – Mode of transaction
Customer Rating – Customer feedback score
🌟 Unique Points
